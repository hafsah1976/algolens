# Third-Party Notices

AlgoLens uses a small set of third-party frontend libraries for graph and node-based learning visuals.

## Cytoscape.js

- Package: `cytoscape`
- Use: Interactive graph canvas for BFS and DFS exploration.
- License: MIT
- Notes: AlgoLens uses custom local graph data and does not load third-party graph datasets.

## React Flow / xyflow

- Package: `@xyflow/react`
- Use: Reserved for future node-based tree, graph, and relationship visualizations.
- License: MIT
- Notes: Keep any future React Flow visuals backed by first-party curriculum data or clearly attributed content.

Before adding new visualization libraries, confirm the license, keep dependency scope small, and add the package here.
