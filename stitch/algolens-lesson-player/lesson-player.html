Pattern: Two Pointers
Concept Introduction

Convergence Logic

The Two Pointers technique is a fundamental pattern used to search for pairs in a sorted array. By initializing pointers at both ends of the collection, we can systematically eliminate possibilities and narrow our search space in linear time, O(n).

Rather than using a nested loop which results in O(n²) complexity, we leverage the sorted nature of the data to move our pointers inward based on logical conditions. 

function findPair(arr, target) {
  let left = 0;
  let right = arr.length - 1;

  while (left < right) {
    let sum = arr[left] + arr[right];
    
    if (sum === target) return [left, right];
    if (sum < target) left++;
    else right--;
  }
  return null;
}
